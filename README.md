## About 
This is a watchlist webapp to keep track of what dramas, movies, or series you have watched and would like to watch. Though this webapp is mainly Korean and Chinese Drama Themed, you can still utilize it to capture other films, dramas too!
You can edit your entries, move them from one list to other and navigated easily from one webpage to another using the menu button on the top-right corner. 
## Tools Used
* VS Code as IDE
* EJS and JS for code functionality
* CSS for styling
* Canva for creating the website background
* MongoDB for storing the entries
* Github Co-pilot for bug fixing and a little bit of styling
